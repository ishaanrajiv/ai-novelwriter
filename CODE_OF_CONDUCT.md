# Code of Conduct

This project is committed to a respectful, harassment-free community for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Being respectful in communication.
- Accepting constructive feedback.
- Focusing on what is best for the project and community.

Examples of unacceptable behavior include:

- Harassment, insults, or personal attacks.
- Discrimination in any form.
- Publishing private information without permission.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct and may remove, edit, or reject comments, commits, code, issues, and other contributions that do not align with these standards.

## Reporting

To report unacceptable behavior, open a private security report or contact the maintainer directly through repository contact channels.
